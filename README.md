# Image Quality Metrics
This library provides Vulkan implementations of SSIM, M-SVD, FSIM, FLIP
image quality metrics.

## Prerequisites
- C++ 20
- Vulkan 1.2+
- OpenCV (only for M-SVD)
- tbb (Thread Building Blocks) (only for M-SVD)
- VkFFT (only for FSIM)

## Usage
Main executable is `IQM`. 

## Implemented methods
### SSIM
### M-SVD
### FSIM
### FLIP
